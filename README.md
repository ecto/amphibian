pipe an ssh session to your browser with socket.io

##install

    npm install -g amphibian

##tap on some keys

````
amp -h
amp myServer -p 3000 &
````

##there's a catch

this only supports passwordless connections, so you have to have your `~/.ssh/config` set up properly
